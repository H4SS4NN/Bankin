

1. Naviguer dans le répertoire frontend :
   ```
   cd bankin-app
   ```

2. Installer les dépendances :

   ```
   npm install
   ```

3. Lancer l'application frontend :
   ```
   npm run dev
   ```

4. Ouvrir votre navigateur et naviguer vers :
   ```
   http://localhost:3000
   ```

### Structure des fichiers :
- src/
  - App.jsx : Composant React principal
  - App.css : Styles pour le composant React
- package.json : Le fichier de configuration des dépendances pour l'application React
