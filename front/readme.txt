1. Cloner le dépôt :
   ```
   git clone <repository-url>
   cd <repository-directory>
   ```

2. Naviguer dans le répertoire frontend :
   ```
   cd bankin-app
   ```

3. Installer les dépendances :
   ```
   npm install
   ```

4. Lancer l'application frontend :
   ```
   npm run dev
   ```

5. Ouvrir votre navigateur et naviguer vers :
   ```
   http://localhost:3000
   ```

### Structure des fichiers :
- src/
  - App.jsx : Composant React principal
  - App.css : Styles pour le composant React
- package.json : Le fichier de configuration des dépendances pour l'application React
